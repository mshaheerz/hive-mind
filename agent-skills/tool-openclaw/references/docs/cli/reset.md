# `openclaw reset`

Reset local config/state (keeps the CLI installed).

```bash  theme={"theme":{"light":"min-light","dark":"min-dark"}}
openclaw reset
openclaw reset --dry-run
openclaw reset --scope config+creds+sessions --yes --non-interactive
```
