# Next.js + Tailwind CSS + Lucide Starter

A minimal but production‑ready starter that includes:

- **Next.js 13 (App Router compatible)**
- **Tailwind CSS** for utility‑first styling
- **Lucide React** icons
- **Redux Toolkit** for global state management
- Basic **API route** (`/api/auth`) as a placeholder for authentication
- Layout components (`Header`, `Footer`, `Layout`) using Tailwind and Lucide icons
- Ready‑to‑run `package.json` scripts (`dev`, `build`, `start`, `test`)
